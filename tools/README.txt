工具目录说明
============

可以将以下工具文件放置在此目录：

1. 7-Zip (Windows)
   文件名：7z.exe
   下载地址：https://www.7-zip.org/download.html

2. FFmpeg (Windows)
   文件名：ffmpeg.exe
   下载地址：https://ffmpeg.org/download.html

3. 7-Zip (Linux/Mac)
   文件名：7z
   通过包管理器安装：sudo apt install p7zip-full

4. FFmpeg (Linux/Mac)
   文件名：ffmpeg
   通过包管理器安装：sudo apt install ffmpeg

放置在此目录的工具会优先被使用。

快速下载：
- 7-Zip: https://www.7-zip.org/a/7z2301-x64.exe
- FFmpeg: https://github.com/BtbN/FFmpeg-Builds/releases/download/latest/ffmpeg-master-latest-win64-gpl.zip

下载后请将7z.exe和ffmpeg.exe复制到此目录。
