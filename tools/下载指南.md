# 工具下载指南

## 🎯 简单方法

直接将以下文件复制到这个 `tools` 目录或项目根目录：

- **7z.exe** (7-Zip压缩工具)
- **ffmpeg.exe** (FFmpeg视频/图片处理工具)

## 📥 下载地址

### 7-Zip
- 官网下载：https://www.7-zip.org/download.html
- 选择：Windows x64版本
- 下载后解压，找到 `7z.exe` 文件

### FFmpeg
- 官网下载：https://ffmpeg.org/download.html
- 选择：Windows builds from gyan.dev
- 下载：ffmpeg-release-full.7z
- 解压后找到 `bin/ffmpeg.exe` 文件

## 📁 目录结构建议

```
fsdownload/
├── tools/              # 专门存放工具的目录 (推荐)
│   ├── 7z.exe         # 7-Zip可执行文件
│   ├── ffmpeg.exe     # FFmpeg可执行文件
│   └── README.txt     # 本说明文件
├── main.py             # 主程序
├── config.py           # 配置模块
└── ...                # 其他文件
```

或者直接放在项目根目录：

```
fsdownload/
├── 7z.exe              # 7-Zip可执行文件
├── ffmpeg.exe          # FFmpeg可执行文件
├── main.py             # 主程序
├── config.py           # 配置模块
└── ...                # 其他文件
```

## 🔧 使用方法

1. **下载工具文件**
   - 从上述地址下载7-Zip和FFmpeg
   - 解压压缩包

2. **复制文件**
   - 将 `7z.exe` 复制到项目目录
   - 将 `ffmpeg.exe` 复制到项目目录

3. **验证工具**
   - 运行程序，查看工具配置区域
   - 应该显示找到的工具路径

4. **手动配置** (如果自动检测失败)
   - 在程序界面点击"浏览"按钮
   - 手动选择工具文件路径

## ❓ 常见问题

**Q: 为什么需要这些工具？**
A: 7-Zip用于解压压缩文件，FFmpeg用于处理图片和视频。

**Q: 可以不安装这些工具吗？**
A: 不可以，这些是核心功能的必需工具。

**Q: 工具放在哪里最合适？**
A: 推荐放在 `tools` 目录中，这样项目更整洁。

**Q: 如何验证工具是否正常工作？**
A: 运行程序后，在工具配置区域应该显示工具路径。
